== SECONDS

Seconds is a small project that takes short 3 second videos using your device's web cam. The results are hilarious.

Have a play around here:

http://seconds.herokuapp.com/

Key languages and plugins:

* Ruby on Rails

* PandaStream for Rails

* Amazon Web Services

* Javascript

* WebRTC

* Media Stream Recorder

* Backbone.js

* Handlebars.js

* Underscore.js

* Velocity.js

* Pace.js

* SASS
